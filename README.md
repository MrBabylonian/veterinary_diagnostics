# Veterinary Diagnostics

## Requirements

- **Node.js**: Version 25.0.0 or higher is required
  - This project uses Node.js 25+ features and the `@types/node` package is pinned to version ^25.0.3
  - Use `nvm use` to automatically switch to the correct version (see `.nvmrc`)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
